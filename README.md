# java-lotto

> 🎫 로또 미션을 TDD로 진행하는 Repository
    - 구입 금액 입력
        - 1000원 단위,
        - 최대 100000원 제한
    - 1000원 단위로 로또 구매
    - 구매한 로또 개수 출력
    - 6개의 랜덤 숫자 생성
        - 1~45의 수
        - 중복 안됨
    - 구매한 로또 출력
    - 6개의 당첨 번호 입력
        - 1~45의 수
        - 중복 안됨
    - 1개의 보너스 번호 입력
        - 1~45의 수
        - 이전의 6개와 중복 안됨
    - 당첨자 찾기
    - 총 수익률 계산
    - 당첨 통계 출력
    - 총 수익률 출력
        - 정수로 출력
