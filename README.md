# java-lotto

> 🎫 로또 미션을 TDD로 진행하는 Repository

# 로또 요구사항
    - 들여쓰기 1개까지만  
    - else 사용 금지  
    - 메소드 10라인 이하  
    - ArrayList 사용  
    - enum 적용해 프로그래밍 구현  
    - 모든 원시값과 문자열 포장  
    - 축약 금지  
    - 일급 컬렉션 사용  
    
# 로또 기능 구현 목록
    [X] 사용자로부터 구입 금액 입력 받는 기능  
        [X] 1000원 단위 입력  
        [X] 최대 100000원 제한  
    [X] 입력에 따른 로또 구매하는 기능
        [X] 랜덤으로 6개 랜덤 숫자 생성하는 로또 발급
        [X] 1~45의 수, 중복 안됨
    [X] 6개의 당첨 번호를 입력하는 기능
        [X] 1~45의 수, 중복 안됨
    [X] 보너스 볼 입력하는 기능
        [X] 1~45의 수, 중복 안됨
    [ ] 당첨 로또 개수 찾는 기능
    [ ] 총 수익률 계산하는 기능
        [ ] 정수 퍼센트로 출력
